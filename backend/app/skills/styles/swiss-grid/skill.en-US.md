# Swiss Grid

## Color system
- Background: white or very light gray.
- Ink: black, graphite, and neutral gray.
- Accent: red used surgically for hierarchy, markers, or key lines.
- Surface: grid columns, thin rules, and clean modular fields.

## Visual character
Rational, typographic, precise, and disciplined. Beauty comes from alignment and proportion.

## Typography
Use flush-left sans-serif typography, strong scale hierarchy, and consistent baseline rhythm. Avoid centered decorative titles.

## Layout language
Use strict columns, visible or implied grids, modular spacing, numbered sections, and clear reading paths. Every element must align to a system.

## Data and shape language
Use clean tables, grid-based comparisons, red markers, thin axes, and exact labels. Avoid ornamental shapes.

## Best for
Design analysis, information architecture, serious reports, product systems, and precise explanatory decks.

## Avoid
Freeform placement, soft gradients, random illustration, decorative curves, and inconsistent spacing.
