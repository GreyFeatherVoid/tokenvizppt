# Tech Signal

## Color system
- Background: dark navy, mist gray, or subtle blue-to-slate gradient.
- Ink: near-white on dark pages or deep slate on light pages.
- Accent: cyan, electric blue, or signal green, used as lines and highlights.
- Surface: translucent panels, technical cards, and low-opacity grid fields.

## Visual character
Sharp, modular, product-keynote energy. It should feel engineered and precise, not generic SaaS decoration.

## Typography
Use modern sans-serif or light technical typography. Headlines should be clear and geometric. Body copy should be brief and label-like.

## Layout language
Use modular grids, product panels, signal paths, stacked cards, system diagrams, architecture blocks, and controlled depth. Use diagonal or radial energy only when it reinforces the narrative.

## Data and shape language
Use metric chips, status indicators, flow lines, node diagrams, small technical labels, and thin luminous rules. Keep glow restrained and readable.

## Best for
AI, software, infrastructure, security, product launches, platform strategy, and future-facing narratives.

## Avoid
Purple default gradients, excessive glow, random circuit-board decoration, unreadable dark slides, and meaningless futuristic icons.
